Installation
============

Requirements
------------

- Python 3.11+
- streamlit
- pydeck
- pyzmq

Install the application
-----------------------

Using your package manager of choice, install the project and its runtime dependencies.

Examples:

.. code-block:: bash

   pip install -e .

or:

.. code-block:: bash

   uv sync

Install documentation dependencies
----------------------------------

.. code-block:: bash

   pip install -e .[docs]

If you do not use optional dependencies, install Sphinx directly:

.. code-block:: bash

   pip install sphinx
